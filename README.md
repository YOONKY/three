# what is Three.js?
Three.js를 학습한 레포입니다.

## how to run?
쉽고 빠른 번들링을 위해 parcel을 사용했습니다.

```
<!-- at three directory  -->
yarn parcel src/index.html
```

리스트를 클릭하여 원하는 테마의 결과물을 확인할 수 있습니다.
